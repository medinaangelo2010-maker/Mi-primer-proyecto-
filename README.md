# Mi-primer-proyecto-
Calculadora interactiva desarrollada con HTML, CSS y JavaScript que soporta operaciones básicas y funciones científicas."
Características
Operaciones Básicas: Suma, resta, multiplicación y división.
Funciones Científicas: Potencias, raíces cuadradas, seno, coseno y logaritmos.
Interfaz Responsiva: Diseñada para verse bien en móviles y computadoras.
Historial: (Opcional) Guarda tus últimos cálculos.
🛠️ Tecnologías utilizadas
HTML5: Estructura de la calculadora.
CSS3: Diseño moderno y botones interactivos.
JavaScript (ES6): Toda la lógica matemática
